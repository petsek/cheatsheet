CSS selectors

Definition

CSS selectors define the elements to which a set of CSS rules apply.


Basic selectors

Selector Name Description Syntax Example
Universal selector Selects all elements. Optionally, it may be restricted to a specific namespace or to all namespaces * ns|* *|* *  will match all elements of the document
Type selector Selects all elements that have the given node name elementname input  will match any <input> element
Class selector Selects all elements that have the given class attribute. .classname .index  will match any element that has a class of "index"
ID selector Selects an element based on the value of its id attribute. There should be only one element with a given ID in a document #idname #toc  will match any element that has the ID "toc"
Attribute selector Selects all elements that have the given attribute. [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value] [autoplay]  will match all elements that have the autoplay  set (to any value)
